# project

Bot em Python que automatiza tarefas no SBSWEB usando PyAutoGUI.

## Configuration

1. Crie um arquivo `.env` na raiz do projeto com as seguintes variáveis:
    ```plaintext
    login = login
    password = password
    ```

2. Instale as dependências:
    ```sh
    pip install -r requirements.txt
    ```

3. Execute o bot:
    ```sh
    python -m bot.main
    ```
